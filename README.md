# Install Tesseract 4 In Ubuntu

```
sudo apt install tesseract-ocr
```

# Older Ubuntu Versions

```
sudo add-apt-repository ppa:alex-p/tesseract-ocr
sudo apt-get update
sudo apt install tesseract-ocr
```
